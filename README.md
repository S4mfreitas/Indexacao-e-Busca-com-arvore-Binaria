# Indexa-o-e-Busca-com-rvore-Bin-ria
Exercício de AEDS II (UNIMONTES). Implementação de sistema de cadastro de clientes com persistência em arquivo binário e indexação via Árvore Binária de Busca. O programa mapeia chaves e posições do arquivo na árvore para realizar buscas otimizadas e listagem ordenada (In-Order).
